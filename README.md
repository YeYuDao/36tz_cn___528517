<br/></br>### 课程介绍:
近年来人工智能如火如荼，如何在人工智能大浪潮中为自己人生赢取更多的砝码，成为众多IT从业者所关心和关注的问题。自然语言处理（NLP）作为人工智能重要的研究热门方向之一，吸引着众多NLP学习爱好者，本课程是作为进去NLP领域的第一步，希望能给入门者提供一些帮助。由于国内的从业环境大多以处理中文为主，因此本套课程也是以讲解中文分词算法为主。
